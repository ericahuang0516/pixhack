## 去叨位

結合Pixnet旅遊相關文章和Instagram打卡資訊，透過與chatbot的人機互動，根據您的目標及興趣偏好推薦個人化的在地旅遊行程。使用者可依據推薦行程再自行編輯調整行程順序。
